# dotfiles

Just my dotfiles for macOS, kept on GitHub for easy access when switching devices. Use at your own risk.

## requirements

1. homebrew
2. tmux
3. neovim
4. nerd font
5. node.js
6. im-select
7. lazygit
8. ripgrep
9. fd
10. else...

## operations

```shell
source setup.sh
```
```shell
source uninstall.sh
```
```shell
source load.sh
```
